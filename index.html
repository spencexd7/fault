Getting Started

  1. Add fault to your discord server.
  2. Join fault's discord support server for more help Discord server.
  3. Type $help to see a list of available commands.
  4. Type $help to see more information about a command.
  5. Type $setprefix to change the bot server prefix.

Utility Commands

Command Description Example
$help [command] Shows list of all available commands. $help ping
